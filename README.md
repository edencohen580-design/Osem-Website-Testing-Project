# Osem-Website-Testing-Project
##  Project Overview
Comprehensive quality assurance testing project for Osem's official website, including competitive analysis against Unilever and Strauss brands.
**Duration:** May 2024  
**Team Size:** 3 QA Testers (Alpha Team)  
**Role:** Manual QA Tester
--------
##  Testing Scope
### Testing Types Performed:
-  **Functionality Testing** - Verified all features work as expected
-  **GUI Testing** - Visual consistency across pages
-  **Usability Testing** - User experience evaluation
-  **Performance Testing** - Load times and responsiveness
-  **Compatibility Testing** - Cross-browser testing (Chrome, Firefox, Edge, Safari)
-  **Accessibility Testing** - WCAG compliance checks
-  **Security Testing** - Basic security vulnerability checks
-  **CRUD Testing** - Database operations validation
-  **Survival & Recovery** - System resilience testing
##  Key Results

- **Total Bugs Identified:** 15+
- **Bug Distribution:**
  - Functionality: 32.4%
  - GUI: 18.5%
  - Compatibility: 11.1%
  - Performance: 10.2%
  - Usability: 18.5%
  - Accessibility: 8.3%
  - - **Test Coverage:**
  - GUI: 70%
  - Functionality: 90%

- **Browsers Tested:** Firefox, Chrome, Microsoft Edge, Safari
